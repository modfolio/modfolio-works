# docs/updates/

이 디렉토리는 **생태계 전체에 공유할 만한 발견/제안**을 기록하는 곳.

modfolio-universe가 주기적으로 이 디렉토리를 수집하여 생태계 지식에 통합한다.

## 작성 가이드

`.claude/skills/updates.md` 참조 또는 `/updates` skill 호출.

## 카테고리

- **gotcha**: 다른 앱도 빠질 수 있는 함정/주의사항
- **pattern**: 생태계 전체에 적용할 만한 모범 사례
- **suggestion**: 생태계 규칙/템플릿 변경 제안
- **question**: universe에서 판단이 필요한 사항

## 파일 형식

```
YYYYMMDD-slug.md
```
